Virtual Office 6.2 July 2018. What's NEW.

  • POWER SEARCH

    You can now SEARCH using the following fields: Master Bed Location, Laundry Location and Parking Type. To SEARCH for Number of Parking Spaces use Total Parking Spaces. We also now support up to 3 HOA Fees, when these are available in the MLS. Also added functionality to support BUILDER and BUILDER2.

  • FULL VIEW

    Numerous enhancements have been made to FULL VIEW. These include the email address being shown in full (no longer truncated). The Complex Name and End Unit have been added. We improved readability, especially when fields overlapped.

  • QUICKMAIL

    The system will now remember the last 20 emails you used, not just the past few. We also fixed a bug that prevented you from changing THEMES.

  • AGENT PHONE NUMBERS

    All Agent Phone Numbers, including the Direct Phone Number, are available from LIST VIEW. Click the ‘Agent Info’ button on the toolbar - it is the 5th button from the left.

  • EASYCMA

    Fixed an issue in EasyCMA where entering a user name that was too long or that contained funky characters could cause the program to crash.

  • EMAIL ALERTS

    EMAIL ALERTS now runs 3 times per day - morning, noon and at night.

  • NEW INSTALL PROGRAM

    New INSTALL program that supports Windows 10 and earlier.
